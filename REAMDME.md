# Linux Commands

A beginner-friendly repository containing commonly used Linux commands with examples and explanations.

This repository is created to help students, developers, and Linux beginners learn essential terminal commands used in daily development and system operations.

## Topics Covered

- File & Directory Commands
- File Permissions
- Process Management
- Networking Commands
- Disk & Memory Commands
- Package Management
- User Management
- Search & Text Processing Commands

## Basic Commands

### Navigation Commands

```bash
pwd
```
Show current directory.

```bash
ls
```
List files and directories.

```bash
ls -la
```
List all files including hidden files.

```bash
cd folder_name
```
Move into a directory.

```bash
cd ..
```
Move back one directory.

---

### File Commands

```bash
touch file.txt
```
Create a new file.

```bash
mkdir folder_name
```
Create a new directory.

```bash
rm file.txt
```
Delete a file.

```bash
rm -rf folder_name
```
Delete a directory.

```bash
cp file1 file2
```
Copy a file.

```bash
mv file1 file2
```
Move or rename a file.

---

### File Viewing

```bash
cat file.txt
```
Display file content.

```bash
less file.txt
```
View large files.

```bash
head file.txt
```
Show first lines.

```bash
tail file.txt
```
Show last lines.

---

### Permissions

```bash
chmod 755 file.sh
```
Change file permissions.

```bash
chown user:user file.txt
```
Change ownership.

---

### Process Commands

```bash
ps
```
Show running processes.

```bash
top
```
Monitor system processes.

```bash
kill PID
```
Terminate a process.

---

### Network Commands

```bash
ping google.com
```
Check internet connectivity.

```bash
ifconfig
```
Show network info.

```bash
netstat -tuln
```
Show active ports.
